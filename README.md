# DIY Kiosk - Releases

**DIY Kiosk** is a POS/Kiosk system for F&B businesses in Vietnam.

## Download

Download the latest APK from the [Releases](../../releases) page.

## Features

- Offline-first - works without internet
- QR payment via payOS (VietQR)
- Easy menu management
